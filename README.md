## Managing-Big-Data-with-MySQL
This course was completed on Coursera in my free time as I aim to practise and understand SQL syntax more in-depthly. 
The files in this repository are practise questions that allowed me to generate queries from scratch to extract the relevant data or generate the required table of data to satisfy the business query. The database used is from a company called Dognition with different breeds of dogs that performed behavioural tests to categorize or label them with a personality trait.
